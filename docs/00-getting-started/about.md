# 关于本项目

## Maintainers

本项目由以下成员发起与维护：

| 角色 | 成员 | 主要职责 |
|------|------|---------|
| 项目发起人 | [@gdmuna](https://github.com/gdmuna) | 架构设计、核心实现、文档体系 |

欢迎通过 [GitHub Issues](https://github.com/gdmuna/nestjs-demo-basic/issues) 反馈问题，或阅读[贡献指南](/guide/contributing)参与共建。

---

## 致谢

本项目站在以下优秀开源项目的肩膀上构建，向它们的作者和社区致以诚挚的感谢：

### 核心框架

| 项目 | 用途 |
|------|------|
| [NestJS](https://nestjs.com) | 后端框架，提供模块化 IoC 容器与装饰器体系 |
| [Prisma](https://www.prisma.io) | 类型安全的数据库 ORM 与迁移工具 |
| [Fastify](https://fastify.dev) | 底层 HTTP 服务器，替代 Express 获得更高性能 |

### 工具链

| 项目 | 用途 |
|------|------|
| [Pino](https://getpino.io) | 结构化 JSON 日志 |
| [Zod](https://zod.dev) | 运行时 Schema 验证 |
| [Passport.js](https://www.passportjs.org) | 认证策略中间件 |
| [jose](https://github.com/panva/jose) | JWT / JWK 操作（ES256） |
| [VitePress](https://vitepress.dev) | 本文档站构建工具 |

### 工程体系

| 项目 | 用途 |
|------|------|
| [dotenvx](https://dotenvx.com) | 多环境 `.env` 管理与加密 |
| [Scalar](https://scalar.com) | 交互式 OpenAPI 文档 UI |
| [pnpm](https://pnpm.io) | 高效包管理 |
